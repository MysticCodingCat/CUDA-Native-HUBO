# CKP Benchmark Instances

52 Cubic Knapsack Problem instances from:

> R. J. Forrester and L. A. Waddell, "Strengthening a linear reformulation of the 0-1 cubic knapsack problem via variable reordering," *J. Comb. Optim.*, vol. 44, no. 1, pp. 498-517, 2022.

Unzip `Instances.zip` to get the 52 `.txt` files (`ckp-{n}-{d}.txt`).
